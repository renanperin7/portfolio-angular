Projeto Stars Wars

Este projeto fiz para praticar requisições de API.

Apresento-lhe um website por mim desenvolvido, que utiliza a API do Star Wars (SWAPI) para obter informações precisas sobre personagens, veículos, planetas e naves. Com essa integração direta à API, você terá acesso a dados atualizados e detalhados do universo de Star Wars.

Projeto MiniBlog

Este Projeto fiz ao final do curso de React. Ele consiste em um site para postagens pessoais.

Podendo criar conta, este site feito com react e usando firebase de backend para armazenar as informações. Serve para realizar postagens pessoais com título, descrição e tags para uma procura mais fácil das postagens. Também é possível comentar nas postagens de outras pessoas.

Projeto Formulário

Este projeto eu fiz para participar de um processo seletivo para a Tinnova, no qual me orgulho bastante.

Esta aplicação utiliza de bibliotecas para formalização dos inputs de cpf e número de telefone, além de usar do localstorage para armazenar as informações dos usuários cadastrados. Podendo criar, editar e excluir na lista de cadastrados.

Projeto Costs

Este projeto, fiz quando estava aprendendo react, para servir de prática. Ele consiste em um site de gerenciamento de orçamentos em projetos.

Podendo criar com nome, valor e categoria do projeto. Também possui ferramentas para edição de cada projeto criado, e foi armazenado em um arquivo json para possíveis atualizações.
