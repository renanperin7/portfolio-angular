Projeto 1

Este projeto, fiz quando estava aprendendo react, para servir de prática. Ele consiste em um site de gerenciamento de orçamentos em projetos.

Podendo criar com nome, valor e categoria do projeto. Também possui ferramentas para edição de cada projeto criado, e foi armazenado em um arquivo json para possíveis atualizações.
