Projeto MiniBlog

Este Projeto fiz ao final do curso de React. Ele consiste em um site para postagens pessoais.

Podendo criar conta, este site feito com react e usando firebase de backend para armazenar as informações. Serve para realizar postagens pessoais com título, descrição e tags para uma procura mais fácil das postagens. Também é possível comentar nas postagens de outras pessoas.
